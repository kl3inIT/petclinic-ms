# Course Deliverables

MSS301 deliverables live in `docs/` beside the harness docs.

## Files

- `docs/SDD_PCMS_v1.0.docx` - software design document.
- `docs/MSS301-SRS.docx` - course SRS deliverable.
- `docs/SRS_PCMS_v1.0.docx` - project SRS deliverable.
- `docs/diagrams/` - generated PNG, Draw.io, D2, and PlantUML assets.
- Existing `docs/*.html` and topical Markdown files are historical or reference
  material for the course package.

## Rule

Treat code as the source of truth when updating deliverables. If the SDD or SRS
describes planned scope that is not implemented, label it planned or
out-of-scope rather than presenting it as current code.
