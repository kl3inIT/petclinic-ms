# Documentation Map

This folder has two kinds of documents:

- Harness docs: living source-of-truth files used by maintainers and agents.
- Course deliverables: DOCX, HTML, diagrams, and generated assets for MSS301.

## Harness Docs

- [vision.md](vision.md) - project intent and scope.
- [roadmap.md](roadmap.md) - progress tracker and backlog.
- [conventions.md](conventions.md) - code and contribution conventions.
- [guidelines/](guidelines/) - reusable platform and workflow mechanics.
- [decisions/](decisions/) - append-only rationale records.
- [specs/](specs/) - current behavior and contracts.
- [tests/](tests/) - coverage matrices.
- [increments/](increments/) - active and completed increment history.
