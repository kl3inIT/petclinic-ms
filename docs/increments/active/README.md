# Active Increments

Put one directory per active increment here:

```text
YYYY-MM-DD-short-slug/
  design.md
  plan.md
```

An active increment design and plan are authoritative only while the increment
is in this folder. Consolidate durable facts into `ARCHITECTURE.md`,
`docs/specs/`, `docs/tests/`, `docs/guidelines/`, and `docs/decisions/` before
moving the directory to `../completed/`.
